    # 快速上手

    引入之前需要使用border-box初始化样式

        *,
    *::before,
    *::after {
        box-sizing: border-box;
    }

    # 按需引入

    在使用的文件中写下以下内容：

       import { Button } from "tk-wheel";

   export default {
       name: "TKButton",
       components: {
           "tk-button": Button
       }
   };

    # 开始使用

    至此，一个基于Vue和tk-wheel的开发环境已经搭建完毕

    各个组件的使用方法请阅读它们各自的文档
